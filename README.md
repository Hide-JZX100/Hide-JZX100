# Hi there, I'm Hidenori Shimizu 👋

---

## ⚡ About Me
- 👨‍💻 **Roots**: プログラミングの原点は小学生時代に学んだ **N88-BASIC**。
- 🚀 **Now**: 数年のブランクを経て **Google Apps Script (GAS)** に再入門。最新の生成AI（Claude/Gemini/GitHub Copilot）をパートナーとして使いこなす **「バイブコーディング」** を実践し、高速かつ柔軟な開発を行っています。
- 🎯 **Mission**: 「データはあるのに活用できていない」「手動のCSV操作に追われている」といった現場のジレンマを、API連携によるフルクラウド化で解消すること。

## 🏆 Key Achievements
- **ネクストエンジンAPI 認証基盤の独自構築**: 
  複雑なOAuth2.0認証フローを独学で解析・突破し、複数プロジェクトで使い回せる「認証ライブラリ」として標準化。
- **データ処理の高速化**: 
  1回1,000行、合計3,000行以上の在庫情報を **約18秒** で同期するバルク処理ロジックの実装。
- **物流・経理業務の自動化**: 
  - 1日3回の手動CSV集計を「1日5回の自動更新」へシフト（出荷予定数・売上集計）。
  - 同梱伝票の紐付け情報をAPIで自動抽出し、運賃削減と事務工数削減の両立を実現。
